## Organización del proyecto

El material está separado por sesión y por tipo de recurso:

```text
sesiones/
	sesion-001/
		clase/
			sesion-001.ipynb
		enunciados/
			ejercicios-sesion-001.ipynb
		pautas/
			pauta-ejercicios-sesion-001.ipynb
```

## Convención sugerida para nuevas sesiones

- Carpeta por sesión: `sesiones/XXX/`
- Notebook de clase: `clase/sesion-XXX.ipynb`
- Notebook de ejercicios/enunciados: `enunciados/ejercicios-sesion-XXX.ipynb`
- Notebook de pauta: `pautas/pauta-ejercicios-sesion-XXX.ipynb`
